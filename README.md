# studentsys
studentsys is a student management system web application developed with php and mysql for managing students, courses, and enrollments.   This system allows admins to add and view students, manage courses, and track student enrollments.

## Features

- **Student Management**
  - Add new students with first name, last name, date of birth, and email.
  - View and manage the list of students.

- **Course Management**
  - Add new courses.
  - View the list of available courses.

- **Enrollment Management**
  - Enroll students in courses.
  - View all enrollments.

- **Dashboard**
  - Quick navigation to students, courses, and enrollments sections.
  - Logout functionality.

---

## Technologies Used

- **Backend:** PHP
- **Frontend:** HTML, Bootstrap 5
- **Database:** MySQL
